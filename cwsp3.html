Types of CSS

Inline CSS

This text is styled with inline CSS.

Inline CSS is used to apply styles directly to an HTML element using the style attribute.

Internal CSS

This text is styled with internal CSS defined within the <style> tag in the <head> section.

Internal CSS is useful when you want to apply styles to a single HTML document.

External CSS

This text would be styled with external CSS .

External CSS involves creating a separate CSS file and linking it to multiple HTML documents.
